

# Skyray Engineering - Corporate Website

A professional, full-featured React-based corporate web application built for **Skyray Engineering**. This platform features a modern UI, product catalog, service details, and a functional quotation/cart system designed to streamline customer inquiries.

## üöÄ Key Features

* **Service & Product Catalogs**: Dynamic browsing of engineering services and product listings with detailed views.
* **Quotation System**: A custom cart and quotation request flow allowing users to select items and submit professional inquiries.
* **Modern UI/UX**: Built with a "Mobile First" approach using Tailwind CSS and Radix UI components for a smooth, accessible experience.
* **User Management**: Includes functional Login and Registration pages to handle customer accounts.
* **Interactive Elements**: Powered by Framer Motion for high-quality page transitions and UI animations.
* **Project Gallery**: A dedicated space to showcase past engineering projects and company expertise.

## üì¶ Getting Started

### Prerequisites

* **Node.js** (v18 or higher recommended)
* **npm** or **yarn**

### Installation

1. **Extract the bundle** and navigate to the project directory:
```bash
cd Skyray_engineering_frontend_React_Version

```


2. **Install dependencies**:
```bash
npm install

```


3. **Start the development server**:
```bash
npm run dev

```


4. **Open your browser** and navigate to `http://localhost:5173` to view the site.

## üèóÔ∏è Project Structure

* `src/pages/`: Contains the main views (Home, About, Services, Cart, etc.).
* `src/components/`: Reusable UI components including the Navigation and Footer.
* `src/data/`: Static data files for products, services, and gallery items for easy content updates.
* `src/styles/`: Global CSS and Tailwind configurations.

## üìù Usage for Customers

* **Requesting a Quote**: Users can add services or products to their cart and navigate to the **Request Quotation** page to provide project details.
* **Navigation**: The top navigation bar provides quick access to the Project Gallery, Services, and Contact information.
